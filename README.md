# Nvidia-pls

Nvidia-pls is a simple EDUCATIONAL Playwright script that parses Amazon product pages to help quickly purchase items.

## Installation

Clone the repository.

```bash
cd /path/to/repository
npm install
touch .env
```

## Environment Variables

Edit your .env file to the following format.

```bash
AMZ_URL=URL to the OFFER LISTING product page
AMZ_EMAIL=youremail
AMZ_PASSWORD=yourpass
AMZ_MAX_PRICE=600.00
```
